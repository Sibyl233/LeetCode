### balabala

「数组」是将相同类型的元素存储于**连续内存空间**的数据结构。

```python
# 遍历
for i in range(n):
        array[i] = xxx

# 索引
array[2]
array[-2]

# 切片
array[:10]
array[1:]
array[::5]
```

「可变数组」基于数组和扩容机制实现，相比普通数组更加灵活。

```python
# 初始化可变数组
array = []

# 尾部添加元素
array.append(2)
```



优势：数组拥有高效的随机访问能力。只要给出下标，就可以用常量时间找到对应元素。（比如二分查找）

劣势：插入、删除元素都会导致大量元素被迫移动，影响效率。

总之适合读操作多，写操作少的场景。
