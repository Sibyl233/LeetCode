## 排序

### 常见排序算法

| 排序算法 | 时间复杂度   | 备注 |
| -------- | ------------ | ---- |
| 插入排序 | $O(n^2)$     |      |
| 选择排序 | $O(n^2)$     |      |
| 冒泡排序 | $O(n^2)$     |      |
| 希尔排序 | $O(nlog(n))$ |      |
| 堆排序   | $O(nlog(n))$ |      |
| 归并排序 | $O(nlog(n))$ |      |
| 快速排序 | $O(nlog(n))$ |      |
| 计数排序 | $O(n)$       |      |
| 基数排序 | $O(n)$       |      |
| 桶排序   | $O(n)$       |      |

Python 中 sort 函数的内部实现机制 为Timsort，是一种结合了归并排序和插入排序的排序算法，最坏时间复杂度为 $O(nlog(n))$。
